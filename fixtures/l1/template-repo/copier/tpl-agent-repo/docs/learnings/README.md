# Learnings

Capture what the agent learns through operation.

## Structure

- `YYYY-MM-DD-topic.md` — dated learning entries
- Link to TIPs if learning should propagate

## Template

```markdown
# [Topic]

## Context
What situation triggered this learning?

## Discovery
What did we learn?

## Evidence
How do we know it's true?

## Application
Where else does this apply?

## TIP Candidate
Should this become a TIP? Why/why not?
```

## Propagation

Agent learnings that generalize should be proposed as TIPs:
- Domain learnings → L1 domain TIPs
- Meta learnings → L0 meta TIPs
