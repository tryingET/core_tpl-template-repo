# Learnings

Capture organizational learnings about governance, culture, and operations.

## Structure

- `YYYY-MM-DD-topic.md` — dated learning entries
- Link to TIPs if learning should propagate

## Template

```markdown
# [Topic]

## Context
What situation triggered this learning?

## Discovery
What did we learn?

## Evidence
How do we know it's true?

## Application
Where else does this apply?

## TIP Candidate
Should this become a TIP? Why/why not?
```

## Propagation

Organizational learnings that apply across companies should be proposed as meta-TIPs to L0.
