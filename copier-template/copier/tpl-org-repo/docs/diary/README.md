# Diary

Session-by-session capture for organizational knowledge crystallization.

## Purpose

Raw experience → Crystallized learnings → Policy/governance updates

**Knowledge that isn't crystallized is knowledge that will be re-learned the hard way.**

## Entry Template

```markdown
# YYYY-MM-DD — [Session Focus]

## What Happened
- [Events, decisions, changes]

## What Surprised Us
- [Unexpected outcomes]

## Organizational Patterns
- [Repeated behaviors, cultural observations]

## Policy Implications
- [What might need to change]

## Crystallization Candidates

### → docs/learnings/
- [Learning to crystallize]

### → governance/
- [Policy updates]

### → TIP Proposal
- [If this generalizes to other orgs]
```

## Flow

1. **Capture** during operations
2. **Crystallize** at session end
3. **Propagate** learnings that generalize across organizations
