# Diary

Session-by-session capture for knowledge crystallization.

## Purpose

Raw experience → Crystallized learnings → Propagated TIPs

The diary captures what happened. Learnings extract what it means. TIPs propagate what generalizes.

**Knowledge that isn't crystallized is knowledge that will be re-learned the hard way.**

## Structure

```
docs/diary/
├── YYYY-MM-DD.md    — Daily/session entries
├── YYYY-MM-DD-2.md  — Multiple sessions per day
└── README.md        — This file
```

## Entry Template

```markdown
# YYYY-MM-DD — [Session Focus]

## What I Did
- [Actions taken]

## What Surprised Me
- [Unexpected outcomes, good or bad]

## Patterns That Emerged
- [Repeated structures, behaviors, outcomes]

## Anti-Patterns Found
- [What looked right but was wrong]

## Heuristics Discovered
- [Rules of thumb that proved valid]

## Caveats
- [What doesn't generalize]

## Crystallization Candidates

### → docs/learnings/
- [Learning to crystallize]

### → TIP Proposal
- [TIP candidate with evidence]
```

## Crystallization Flow

1. **During session**: Capture raw in diary
2. **End of session**: Extract crystallization candidates
3. **Weekly review**:
   - Promote diary entries to `docs/learnings/`
   - Propose TIPs for patterns that generalize
4. **Monthly review**: Audit diary → learnings → TIPs pipeline

## Why This Works

- **Capture is cheap**: Don't overthink, just write
- **Crystallization is expensive**: Do it when patterns are clear
- **Propagation compounds**: One TIP benefits all future agents

## Anti-Patterns

- ❌ Skipping diary → learnings die
- ❌ Over-structured diary → friction → skipped
- ❌ Never crystallizing → diary becomes graveyard
- ❌ Crystallizing too early → patterns not yet clear

## The Goal

Every session leaves the agent smarter. Every crystallized learning leaves all agents smarter.
