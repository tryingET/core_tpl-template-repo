# Metrics

Track template health and TIP effectiveness.

## Template Metrics

| Metric | Description | Target |
|--------|-------------|--------|
| L2 generation success | % of clean generations | > 95% |
| TIP acceptance rate | % of TIPs accepted | > 70% |
| TIP propagation | % of meta-TIPs escalated to L0 | Tracked |
| Agent adoption | # of agents using these templates | Growing |

## Measurement Cadence

- Weekly: Generation success rate
- Monthly: TIP review and acceptance
- Quarterly: Cross-L1 propagation analysis
