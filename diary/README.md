# Diary

Repo-local session capture for KES (Knowledge Evolution System).

## Rule

Use `./diary/` in this repository as the canonical raw session log.

- Entry file: `YYYY-MM-DD.md`
- Multiple sessions/day: `YYYY-MM-DD-2.md`
- Crystallize to: `docs/learnings/` and TIPs when patterns generalize

## Entry template

```markdown
# YYYY-MM-DD — [Session Focus]

## What I Did
- [Actions]

## What Surprised Me
- [Unexpected outcomes]

## Patterns
- [Repeated structures]

## Crystallization Candidates
- → docs/learnings/
- → tips/meta/
```
